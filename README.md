# 안경희 소개 웹사이트

신용평가·재무 전문가이자 대학 교수, 『깨진 유리창과 시장의 배신』의 저자
**안경희**님을 소개하는 프리미엄 원페이지 웹사이트입니다. (데스크탑 버전)

## 실행 방법

가장 쉬운 방법 — **`index.html` 을 더블클릭**하면 브라우저에서 바로 열립니다.
(또는 `사이트_열기.bat` 더블클릭)

> 인터넷이 연결돼 있으면 고급 한글 폰트(Noto Serif KR·Pretendard)가 적용되고,
> 오프라인이어도 시스템 기본 폰트로 정상 표시됩니다.

## 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 페이지 구조 (직접 고칠 일 거의 없음) |
| `styles.css` | 디자인·색상·레이아웃 |
| **`data.js`** | **모든 내용(소개·경력·저서·추천사 등) — 여기만 고치면 됩니다** |
| `app.js` | 렌더링 및 챗봇 동작 |
| `사이트_열기.bat` | 실행 파일 |

## 내용 수정 방법 (`data.js`)

메모장이나 편집기로 `data.js` 를 열어 원하는 글자만 바꾸면 됩니다.

- `tagline`, `intro` : 한 줄 소개 / 자기소개
- `stats` : 신뢰 지표 숫자
- `specialties` : 전문분야 카드 (`name`, `desc`)
- `career` : 경력 타임라인 (시간 순). `highlight: true` 는 강조 표시
- `book.chapters` : 저서 목차 — `title`(장 제목) + `sections`(소제목 목록)
- `endorsements` : 추천사 (`name`, `title`, `quote`)
- `teaching.subjects`, `research`, `external` : 강의 과목 / 논문 / 외부 활동
- `qa` : 챗봇 예시 질문·답변 — `{ q: 질문, a: 답변, tags: [검색어들] }`. 항목을 추가하면 챗봇이 그 질문에 답합니다.
- `ownerEmail` : 채우면 챗봇이 못 답한 질문을 이메일로 보내는 버튼이 생깁니다(비워두면 이메일 비공개).

## 챗봇 사용법

- 우측 하단 💬 버튼, 상단 **‘질문’ 메뉴**, 또는 **‘무엇이든 물어보세요’ 섹션**으로 열립니다.
- 예시 질문을 누르거나 자유롭게 입력하면 `data.js`의 정보를 바탕으로 답합니다.
- 챗봇이 모르는 질문이면 “저는 챗봇입니다… 잠시 기다리시면 직접 답해드리겠습니다”를 띄우고 질문을 보관합니다.
  이때 **‘주인이 직접 답변 입력’** 버튼으로 답을 적어 저장하면, 다음부터 같은 질문에는 그 답이 자동으로 표시됩니다(학습).
- 보관된 미답변 질문은 챗봇에 **‘관리자’** 또는 **‘받은 질문’**이라고 입력하면 모아 볼 수 있습니다.
  (별도 ‘관리자 모드’ 화면은 없습니다 — 챗봇에 ‘관리자’라고 입력하는 것이 관리 방법입니다.)
- (참고) 직접 답변·받은 질문은 그 브라우저에만 저장되며, 다른 기기와는 공유되지 않습니다.

## 안경스(경제스터디) 영상 추가·관리

영상을 **모든 기기(데스크탑·모바일)에 영구 반영**하려면 아래 도구를 쓰세요.
(브라우저에만 저장하는 방식은 다른 기기로 넘어가지 않아 사용하지 않습니다.)

1. **`안경스_관리.bat` 더블클릭** → 관리 화면이 브라우저에 열립니다.
2. 영상 추가 — 두 가지 방법:
   - **유튜브 링크**(youtu.be/… 또는 watch?v=…) + 제목 → **[추가]** (권장, 용량·속도 최적)
   - **동영상 파일** → **[🎬 동영상 파일 올리기]** 로 PC 파일을 사이트에 직접 호스팅
     (100MB 미만 권장. 큰 영상은 유튜브가 빠릅니다.)
3. 목록에서 순서 조정(▲▼)·제목 수정·삭제가 가능합니다.
4. **[게시하기]** 를 누르면 `videos.js`(와 올린 파일)가 GitHub에 자동 업로드됩니다.
   → **1~2분 뒤** 사이트(https://akhbeing-design.github.io/ahn-kyunghee/)에 반영됩니다.

> 원리: 영상 목록은 `videos.js` 파일에 저장되고, 이 파일이 GitHub에 올라가 모두에게 보입니다.
> 관리 도구(`안경스_관리.py`)는 내 PC에서만 도는 개인용이며, 게시에는 이미 로그인된 Git 인증을 씁니다.

## 질의응답 · 방문자 질문 받기

- 메뉴 마지막 **‘질의응답’** 섹션에서 방문자가 질문을 남길 수 있습니다.
  질문은 **akh.being@gmail.com 으로 자동 전송**됩니다(이메일 주소는 화면에 노출 안 됨).
  방문자가 ‘답변받을 이메일’을 남기면 그 메일로 **직접 답장**하시면 됩니다.
- **공개/비밀 답변 관리** — `질의응답_관리.bat` 더블클릭:
  - 질문·답변을 적고 **공개**(사이트 질의응답에 표시) 또는 **비밀**(이 PC에만 저장, 사이트엔 안 보임) 선택
  - **[게시하기]** → 공개 항목이 GitHub에 올라가 1~2분 뒤 사이트에 반영
  - 공개 목록은 `qna.js`, 비밀 기록은 이 PC의 `비밀답변.json`(공개 저장소에 올라가지 않음)
- 자동 전송 키는 `data.js`의 `ownerFormKey`(web3forms), 받는 주소는 `ownerEmail` 입니다.

> ⚠️ 정적 사이트라, 답변을 **특정 방문자 화면으로 되돌릴 수는 없습니다.**
> → 개인 답변은 **이메일 답장**, 여러 사람에게 유용한 답은 **공개 질의응답**으로 올리세요.

## 사진·표지 이미지 넣기 (선택)

1. `images` 폴더를 만들고 사진 파일을 넣습니다.
2. `data.js` 에서
   - 인물 사진: `portrait: null` → `portrait: "images/portrait.jpg"`
   - 책 표지: `book.cover: null` → `cover: "images/book.jpg"`
